# Historical Direct Flights per Airline - Demo

Une application React avec un line chart interactif montrant l'historique des vols directs par compagnie aérienne.

## Fonctionnalités

- **Line Chart interactif** avec 20 compagnies aériennes :
  - 10 Operating Airlines (couleurs normales)
  - 10 Discontinued Airlines (couleurs pastel)
- **Masquage individuel** : Cliquez sur un élément de la légende pour le masquer/afficher
- **Masquage par groupe** : Boutons pour masquer/afficher tout un groupe
- **Highlight au survol** : Survolez une ligne ou un élément de la légende pour mettre en évidence cette ligne et assombrir les autres

## Installation

```bash
npm install
```

## Lancement

```bash
npm run dev
```

L'application sera accessible sur `http://localhost:5173`

## Technologies utilisées

- React 18
- TypeScript
- Vite
- Recharts (bibliothèque de charts pour React)
